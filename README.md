# bullet-project How to Use

## repositoryをローカルにclone
### cd ~ & git clone https://github.com/tadayuki-inoue/bullet-project.git

## bullet-projectに移動
### cd bullet-project

## build
### docker-compose up -d --build

## コンテナに入る
### docker-compose exec bullet-project bash

## テストコード実行
### python work/index.py
